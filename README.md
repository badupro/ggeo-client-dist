# GGeo Client

Mobile GPS Location Spoofer.

## Install

```
git clone https://github.com/badupro/ggeo-client-dist ggeo-client
cd ggeo-client
```

- **macOS:** double-click `GGeo.command`
- **Windows:** right-click `GGeo.bat` → **Run as administrator**

A menu opens with: Setup, Start server, View log, Update, Uninstall.

Contact your administrator for credentials.
